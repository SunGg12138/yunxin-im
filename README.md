# yunxin-im

## 安装

```bash
$ npm install yunxin-im
```

## 使用

```javascript
const YunxinIM = require('node-yunxin-im');
const yunxinIM = new YunxinIM('<你的AppKey>', '<你的AppSecret>');

// 返回的是Promise
// 具体的接口和参数：http://dev.yunxin.163.com/docs/product/IM即时通讯/新手接入指南
// 链接页面 > 服务端API文档
yunxinIM.exec('user/create.action', {
  accid: accid,
  donnopOpen: false
});
```
